# End-to-End Data Engineering Project

## Overview
This project demonstrates a complete data engineering pipeline from raw data ingestion to cleaned data analysis.

## Pipeline Flow
1. Ingest raw sales data
2. Validate and clean invalid records
3. Transform data and create metrics
4. Generate business insights using SQL

## Key Insights
- Revenue by region
- Monthly revenue trends
- Top-selling products

## Skills Used
- Python
- Pandas
- SQL
- Data Cleaning
- Data Validation
- Pipeline Design
It is created as a demo portfolio project to showcase relevant skills.
